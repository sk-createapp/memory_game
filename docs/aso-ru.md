# ASO 設計書（ロシア App Store・確定版）

> **編集者向けサマリー（日本語）**
> - 推奨タイトル: **「Тренировка памяти: фотопамять」**（記憶力トレーニング：写真記憶）。最重要キーワード `тренировка памяти`（記憶力トレーニング）を完全一致で押さえつつ、差別化軸 `фотопамять`（写真記憶＝фотографическая память の口語短縮）を立てる。
> - 推奨サブタイトル: **「Зрительная память и внимание」**（視覚記憶と注意力／28字）。Wikium 等が取りきれていない `зрительная память`（視覚記憶）を新規キーワードとして確保し、汎用の `внимание` も回収。
> - 採用理由: ロシアの強競合（Викиум / CogniFit / NeuroNation）は「汎用脳トレ（память・внимание・мышление）」に集中。本アプリは **「фотопамять × зрительная память × かわいい相棒育成 × シニアにも優しい」** で空き地を取る。汎用ボリューム語は完全一致で1つだけ確保し、残りの枠を中ボリューム・低競合のニッチ語に回す設計。

ロシア向け App Store の ASO（App Store 最適化）確定版。App Store Connect の **ロシア語（ru）ローカリゼーション** にそのまま転記して使う。

- 対象ストア: App Store（ロシア / ru ロケール）
- 差別化軸: `фотографическая память`（写真記憶）× `зрительная память`（視覚記憶）× `相棒の育成`
- メインペルソナ: 物忘れ・加齢が気になる 40〜60代＋アクティブシニア（пожилые / люди старшего возраста）
- サブペルソナ: 記憶力・集中力を伸ばしたい学生・社会人（студенты・взрослые）
- 主競合: **Викиум（Wikium）**＝「Тренажеры для развития мозга」。他に CogniFit / NeuroNation / Peak / Lumosity / Memoris。

## 1. タイトル（30字以内）

```
Тренировка памяти: фотопамять
```

- 文字数: **29字**（キリル文字・スペース・記号すべて1字カウント）
- カバー語: `тренировка`（トレーニング）/ `память`（記憶）/ `тренировка памяти`（完全一致＝最重要・最高ボリューム）/ `фотопамять`（写真記憶＝口語短縮・差別化軸）
- 補足: `фотопамять` は `фотографическая память` の自然な短縮形でロシア語話者が実際に検索する語。タイトルに `фото` を置くことで `фото`＋`память` の組み合わせ索引も狙える。

## 2. サブタイトル（30字以内）

```
Зрительная память и внимание
```

- 文字数: **28字**
- カバー語（タイトル未使用の新語のみ）: `зрительная`（視覚の）/ `зрительная память`（視覚記憶＝低競合の主要ニッチ語・完全一致）/ `внимание`（注意力・集中力＝汎用高ボリューム）
- 補足: `мозг`（脳）はサブタイトルに入れず、後述キーワード欄で `мозга` を別途回収（Apple は `тренировка`＋`мозга` を自動合成しないため、キーワード欄に `мозга` を明示する）。
- 不採用案メモ: 「Зрительная память, внимание, мозг»」は **32字で上限超過**のため不採用。

## 3. キーワードフィールド（ロシア語 ru ロケール・100字以内）

半角カンマ区切り・スペースなし・既出語の重複なし・複合語は単語分割。

```
эйдетическая,кратковременная,мозга,развитие,концентрация,мнемотехника,запоминание,пазл,головоломка
```

- 文字数: **98字**（上限100字。バッファ2字）
- 補足: ロシア語は1語が長い（`фотографическая`=15字、`кратковременная`=15字）ため、100字枠に収まるのは実質9語前後。枠が貴重なので「タイトル/サブタイトルで取れない高価値語」のみを厳選した。
- 設計意図:
  - `эйдетическая` `кратковременная`：`память` と自動合成され `эйдетическая память`（直観像記憶＝фотографическая память の学術同義）/ `кратковременная память`（短期記憶）を回収する低競合ニッチ。差別化軸の補強。
  - `мозга` `развитие`：`развитие мозга`（脳の発達）/ `тренировка мозга`（脳トレ）を分割で回収する汎用高ボリューム。
  - `концентрация`：集中力（`внимание` はサブタイトルで取得済みのため別語で補完）。
  - `мнемотехника` `запоминание`：記憶術・暗記の実需要語。
  - `пазл` `головоломка`：パズル/謎解き検索の回収（パズルカテゴリ流入）。
- 既出語の重複回避: `фотопамять`（タイトル）/ `зрительная` `память` `внимание`（サブタイトル）はキーワード欄に再掲しない。`фотографическая` は `фотопамять`（タイトル）が同義意図を回収するため、長さ優先で枠から外した。
- 枠外に置いた次点語（リリース後の差し替え候補）: `визуальная` `пожилых` `взрослых` `зарядка` `гимнастика` `нейробика` `фокус` `скорость` `логика` `бесплатно`。
- 注意: `деменция`（認知症）/ `Альцгеймер`（アルツハイマー）等の医療・症状語は **可視/索引フィールドに入れない**（審査リスク・効能断定回避）。上記キーワード群は症状語を含まない安全設計。

## 4. 関連ロケール / 綴りメモ

- **ru ロケールのみが索引対象**: ロシア App Store の標準言語はロシア語。英語(US)キーワードの追加索引は日本ストアほど明確でないため、まず ru に全振りする。将来 CIS（カザフ等）展開時に en も併設検討。
- **綴り揺れ**: `фотопамять`（短縮・タイトル採用）/ `фотографическая память`（正式・キーワード分割で回収）。両方を別フィールドで押さえる設計。
- **キリル文字 `й` `ё`**: `ё` は実検索で `е` と打たれることが多い。本設計では `ё` を含む主要語を使用していないため問題なし（例: 不使用）。
- **格変化（падежи）**: ロシア語は格変化するが、Apple の正規化により語幹一致である程度吸収される。キーワードは主格（именительный）または生格（`мозга`＝`развитие мозга` の自然形）で登録。
- **「写真記憶」概念**: ロシアでは `фотографическая память` が一般語、学術寄りは `эйдетическая память`。両方を確保済み。

## 8. 実装メモ

- タイトル・サブタイトルは **App Store Connect のロケール別「ローカリゼーション」（ru）** で設定。バイナリ内の `CFBundleDisplayName` とは独立のためコード変更不要。
- タイトル / サブタイトル / キーワードの変更は**バージョンアップ申請時のみ**反映。プロモーションテキストは随時変更可。
- カテゴリ候補: **Образование（教育）** または **Игры > Головоломки（ゲーム＞パズル）**。脳トレ系の定石。競合 Викиум は「Образование」。リリース後に計測して入替検討。
- 医療効果表現（`деменция` / `Альцгеймер` / `лечение` 等）は可視・索引フィールドで使用しない（審査リスク・効能断定回避）。本設計は症状語ゼロ。
- サブタイトルとプロモテキストは初稿で上限超過したため**確定版に短縮済み**。転記時は必ず確定版（28字 / 152字）を使う。
- ロシア向け価格はローカル最低価格帯相当に設定。説明文・サブスク開示の `<местная цена>` を実価格に置換すること。
- リリース後は実検索順位を見て、効いていない語を差し替える**キーワードサイクル**を回す（初期設計は仮説）。特に汎用語 `внимание` `концентрация` は競合が強いため、効果が薄ければニッチ語へ寄せる。
