# ASO 設計書（中国語 App Store・確定版）

記憶力トレーニングゲーム「照相记忆（写真記憶）」の **中国語ストア** 向け ASO（App Store 最適化）確定版。
App Store Connect の **简体中文（zh-Hans）** ロケールをそのまま転記して使う。繁体字（zh-Hant：台湾・香港）の差し替え案も第4章に併記。

- 対象ストア: App Store 中国本土（简体）／台湾・香港（繁体）
- 差別化軸: `照相记忆（写真記憶）` × `看图记忆（短期視覚記憶）` × `相棒动物の育成` × `中老年にもやさしいシンプルさ`
- メインペルソナ: 物忘れ・ボケ予防が気になる 40〜60代＋アクティブシニア（中老年・銀髪族）
- サブペルソナ: 記憶力・集中力・ワーキングメモリを伸ばしたい学生・社会人

## 1. タイトル（30字以内 ※漢字も1字=1カウント）

```
记忆力训练-照相记忆脑力训练益智游戏
```

- 文字数: **18字**（`-` は半角 1 カウント、漢字 17 字）
- カバー語（完全一致）: `记忆力` / `记忆力训练`（最高ボリューム語）/ `照相记忆`（差別化・低競合）/ `脑力` / `脑力训练` / `益智游戏`（パズル定番）/ `训练`
- 自動合成: `记忆`＋`脑力`＋`训练`＋`游戏` の組合せで `记忆训练` `记忆游戏` `脑力游戏` 等もカバーされやすい。
- 設計意図: 中国ストア競合（例「记忆大作战:卡片配对,记忆力训练,专注力训练」）は最重要枠に高ボリューム語をカンマ羅列するのが定石。本案は最左に最高権重語 `记忆力训练` を置き、中央に唯一の差別化語 `照相记忆` を挟む。

```
看图记忆练专注力·过目不忘·中老年防健忘脑锻炼
```

- 文字数: **23字**（`·` は中点 1 カウント ×2 を含む、漢字 21 字）
- カバー語（タイトル未使用の新語のみ）: `看图记忆`（→`看图`/`记忆`合成）/ `专注力`（高ボリューム、競合 `专注力训练` を自動合成）/ `过目不忘`（写真記憶の口語・高アスピレーション）/ `中老年`（シニア需要）/ `防健忘`（物忘れ対策・効能断定を避けた表現）/ `脑锻炼`（`脑`＋`锻炼`）

## 9. 実装メモ

- タイトル・サブタイトルは **App Store Connect のロケール別「本地化（ローカリゼーション）」** で設定する。バイナリ内の `CFBundleDisplayName=Memory Game` とは独立のため、コード変更は不要。
- タイトル / サブタイトル / キーワードの変更は **バージョンアップ申請時のみ** 反映。プロモーションテキストは随時変更可。
- カテゴリ候補: **益智解谜（パズル）** または **教育**。脳トレ系は両カテゴリに分散。直接競合「记忆大作战」は益智解谜、「老年游戏」は教育/休闲。リリース後に計測して入替検討。
- **医療効果表現の断定回避**: `防痴呆`/`治疗阿尔茨海默`/`治愈健忘症` 等の断定は中国の広告法（虚偽・誇大広告）および Apple 審査でリスク。可視フィールドでは `防健忘`/`护脑`/`脑锻炼`/`认知训练`/`脑年龄` 等の **習慣・トレーニング表現** に留める。本書の可視フィールドはこの方針で記述済み。
- **简繁の文字数注意**: CJK は簡体字・繁体字とも1字=1カウント。繁体変換で字数が増減しないか必ず再カウント（本書のタイトルは簡18字／繁18字で同数を確認済み）。
- **本土ストアの特性**: 中国本土 App Store はリリース前に ICP 備案（網絡出版/版号）等の規制対応が必要な場合がある。ゲームカテゴリでの本土配信は版号要件に注意（益智系は要確認）。台湾・香港ストアは規制が緩く先行配信しやすい。
- リリース後は実検索順位を見て効いていない語を差し替える **キーワードサイクル** を回す（初期設計は仮説）。特に `舒尔特方格`/`记忆宫殿` 等の高競合語は ASA（Apple Search Ads）データで費用対効果を見て判断。
