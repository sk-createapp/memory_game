# ASO 設計書（韓国 App Store・確定版 / ko）

> **編集者向けサマリー（日本語）**
> - 推奨タイトル: **`기억력 게임 - 사진 기억 두뇌 트레이닝`**（22字）＝「記憶力ゲーム（完全一致の最強需要語）」＋差別化軸「사진 기억（写真記憶）」＋定番検索語「두뇌 트레이닝（脳トレ）」。大手（Lumosity/NeuroNation/실비아）が front-load する `기억력`/`두뇌 트레이닝` を取りつつ、誰も主軸にしていない `사진 기억` を可視化。
> - 推奨サブタイトル: **`위치와 그림을 기억! 집중력·순간기억 훈련`**（23字）＝「位置とイラストを記憶！集中力・瞬間記憶トレーニング」。ゲーム性（位置×内容）を一言で説明しつつ、競合の弱い `순간기억`（瞬間記憶）と `집중력`（集中力）を補完。
> - キーワード欄は `치매`/`어르신`/`시니어`/`건망증`/`공간기억` などシニア需要＋ロングテールで98字。`치매예방` 等の医療断定は可視フィールドで避け、`치매` 単体に留める（注意書きあり）。
> - 想定カテゴリ: **교육（教育）** を主、シニア訴求を強めるなら **건강 및 피트니스（健康/フィットネス）** も検討。

韓国 App Store（ko）向けの ASO（App Store 最適化）確定版。App Store Connect の **韓国語ローカリゼーション** にそのまま転記して使う。

- 対象ストア: App Store（韓国 / 한국어）
- 差別化軸: `사진 기억（포토그래픽 메모리）` × `단기 시각기억` × `귀여운 상棒（相棒）동물 육성`
- メインペルソナ: 物忘れが気になる中高年〜アクティブシニア（두뇌 트레이닝 / 치매 걱정）
- サブペルソナ: 記憶力・集中力を鍛えたい学生・社会人

## 1. タイトル（30字以内 ※ハングルも1字=1カウント）

```
기억력 게임 - 사진 기억 두뇌 트레이닝
```

- 文字数: **22字**（空白・ハイフン含む）
- カバー語: `기억력` / `기억력 게임`（完全一致）/ `사진 기억`（差別化軸）/ `두뇌` / `두뇌 트레이닝` / `게임` / `기억`
- なぜこれか: 韓国の太い実需要は競合が示す通り **기억력게임** と **두뇌 트레이닝**（Lumosity「일일 두뇌 게임」/ NeuroNation「두뇌 트레이닝 과 기억력게임」/ 실비아「치매예방 두뇌 트레이닝 & 기억력게임」）。両方を完全一致で抱えつつ、上位大手が主軸にしていない `사진 기억` を可視フィールドに置いてニッチを取りに行く。

## 2. サブタイトル（30字以内）

```
위치와 그림을 기억! 집중력·순간기억 훈련
```

- 文字数: **23字**（`!` 半角、`·` 全角中黒1字）
- カバー語（タイトル未使用の新語のみ）: `위치`（位置）/ `그림`（イラスト＝視覚要素）/ `집중력`（集中力）/ `순간기억`（瞬間記憶）/ `훈련`（トレーニング）
- Apple は title × subtitle を自動結合・組替えするため、`기억력`(title)＋`훈련`(subtitle) で **`기억력 훈련`**、`두뇌`(title)＋`훈련`(subtitle) で **`두뇌 훈련`** も索引対象になる（→キーワード欄に重ねて入れない）。
- 「位置と内容を覚える」というゲームの核を一文で説明し、CVR と検索の両取りを狙う。

## 3. キーワードフィールド（韓国語ロケール・100字以内）

半角カンマ区切り・スペースなし・タイトル/サブタイトルとの重複なし。

```
치매,순간기억력,두뇌게임,기억력향상,두뇌발달,어르신,시니어,두뇌운동,뇌운동,건망증,암기,관찰력,공간기억,시각기억,단기기억,무료,포토그래픽,기억력강화,뇌자극,머리회전,테스트,향상
```

- 文字数: **98字**（上限100字。バッファあり）／ 22語
- スペースなしで詰めるのは韓国ロケールの定石。Apple はキーワード欄内の単語も自動結合するため、`기억력향상` のように連結しても、`기억력`(title)＋`향상`(kw) など分解索引も働く。`테스트` 単体を残し `기억력`＋`테스트`＝**`기억력 테스트`**（競合「memoryOS」型の実需要）を合成で拾う。
- 注意1: `치매` は医療・症状を示唆する語。`치매예방`（＝「予防」効能の断定）は可視・キーワード両方で**避け**、検索ボリュームの太い `치매` 単体に留める。審査で指摘されたら最優先で削除候補。
- 注意2: `포토그래픽`（포토그래픽 메모리＝写真記憶）は学術的に賛否ある語だが、検索語としては生きている。`사진 기억`（タイトル）と意味は近いが綴りが異なるため別需要として残す。
- 注意3: `어르신`/`시니어`/`건망증` はシニア・物忘れ層の実検索語。差別化（シニアにもやさしい）と合致するため優先度高め。

## 4. 関連メモ（英語併用の是非）

- 韓国ストアは**第1言語=韓国語ロケールのキーワードのみ索引**が基本で、日本のような「英語(US)ロケール併用索引」の確実なメリットは薄い。まずは韓国語100字を埋め切ることを最優先。
- ただしアプリ名に英字ブランド（例: Lumosity, memoryOS）を併記する競合は多い。本アプリは英字社名・サービス名を持たないため、タイトルは純ハングルで統一して可読性とローカル感を優先する。
- 英単語需要（`memory`, `brain training`）を拾いたい場合は、App Store Connect の「英語(U.K.)」など**第2ローカライズを追加設定**して別途キーワードを入れる手がある（運用コストと相談）。初期は不要。

## 8. 実装メモ

- タイトル・サブタイトル・キーワードは **App Store Connect の韓国語「ローカリゼーション」** で設定する。バイナリ内 `CFBundleDisplayName` とは独立のためコード変更は不要。
- タイトル / サブタイトル / キーワードの変更は**バージョンアップ申請時のみ**反映。プロモーションテキストは随時変更可。
- カテゴリ候補: **교육（教育）** を主軸（Lumosity・NeuroNation・memoryOS が採用）。シニア・物忘れ層を強く取りに行くなら **건강 및 피트니스（健康/フィットネス）**（실비아が採用）も検討。リリース後に計測して入替。
- **医療効果の断定回避**: `치매예방`・`인지기능 개선`・`치매 위험 감소` 等は効能の断定（韓国でも表示規制・審査リスク）。実비아は「보건복지부 비의료건강관리 서비스 인증」を根拠に強い表現を使うが、本アプリは未取得のため**同様の断定は使わない**。可視フィールドは `두뇌 운동`・`두뇌 건강 습관` 等の柔らかい表現に留め、キーワードは `치매` 単体まで。
- 説明文の `상棒`（プレースホルダ）は **`동물 친구` / `단짝 친구`** に統一置換（韓国語に「相棒」を1語で表す自然な語がないため）。プロモ/キャプションでは既に `동물 친구`/`친구` を採用済み。
- `<현지 가격>/월` は韓国の実価格に置換（最低価格帯相当）。日本 ¥200/月 と整合する韓国側ティア（例 ₩1,500 前後）を App Store Connect の価格表で確認。
- 文字数注意: ハングルは1字=1カウント。タイトル22字・サブ23字・キーワード98字でいずれも上限内（30/30/100）。語を足す際は再カウント必須。
- リリース後は実検索順位を見て効いていない語を差し替える**キーワードサイクル**を回す（初期設計は仮説）。
